# AI Footwear Product Generator

This project generates product descriptions, bullet points, and SEO titles for footwear products.

## Why this project?
Built to help footwear brands like Walkshark create fast, consistent, and SEO-friendly product content.

## Features
- Product description generation
- Bullet point creation
- SEO title generation

## How to Run

1. Open the file `app.py`
2. Run the script using Python:
   python app.py

## Sample Output

### Wave Luxe (Slides)

Description:
Wave Luxe is a premium Slides designed for comfort and everyday style. It features soft EVA sole and anti-slip grip.

Bullet Points:
- Premium Slides design
- Enhanced comfort with soft EVA sole and anti-slip grip
- Durable and lightweight
- Perfect for daily wear

SEO Title:
Wave Luxe Slides – Comfortable & Stylish | Walkshark
